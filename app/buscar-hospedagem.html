ℹ️ Como ligar os preços reais (APIs gratuitas)

Agora o app já funciona com exemplos e os botões “Ver no site” abrem a busca real de cada site para o seu destino e datas. Para trazer preços e fotos reais para dentro do app, basta conectar uma API gratuita de agregador — a estrutura já está pronta (conectores plugáveis na Edge Function search-stays). Veja o passo a passo em docs/07-busca-hospedagem.md. As APIs oficiais de parceiro entram no futuro como mais um conector, sem mexer no resto.
